# Github contributions analyzer
## Getting started
* Make sure to create a virtual environment if you haven't already
    * Run command (in the terminal of this project directory) `python3 -m venv venv`
* Install the relevant packages. These have a lot of dependencies, so it'll take a moment
    * `pip install -r requirements.txt`
* If you add a new dependcy to this project, update `requirements.txt` with the following:
    * `pip freeze > requirements.txt`
